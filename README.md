# Team Task Manager

## Backend
cd Backend
npm install
Create a `.env` file in the Backend folder:

MONGO_URI=mongodb://127.0.0.1:27017/team-task-manager
JWT_SECRET=mysecretkey

npm run dev

## Frontend
cd Frontend
npm install
npm run dev

## Features
- Login/Register
- Project management
- Task management
- Dashboard
- MongoDB integration
